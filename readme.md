### Install
````

* npm init -y
* npm install --save-dev typescript
* npm install express
* npm install --save-dev @types/express
* npm install --save-dev ts-node nodemon 

````

````
* "start:dev": "nodemon" => JavaScript em modo desenvolvimento;
* "start": "node dist/index.js" => JavaScript em modo normal;
* "build": "tsc" => JavaScript em modo em produção;
````

````
Salvar automatico que fica verificando alterações nos arquivos:
- nodemon
* npm install --save-dev ts-node nodemon
````

### Terminal
````

- node index.js
- node dist/index.js
- npx tsc --watch
- npx ts-node src/index.ts
- npm run start:dev
- npm run build
- npm start

````

### Converter, terminal
````

* npx tsc src/index.ts --outDir dist/
* npx tsc src/index.ts --outDir dist/ --watch
* npx tsc --init --rootDir src --outDir dist --noImplicitAny true --allowJs true

````
